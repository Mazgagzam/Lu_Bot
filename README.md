# Lu Bot

Lu Bot is a mini Telegram bot that sends messages with your text as a quote from [Alua](https://t.me/alua_krr).

## Installation

```bash
git clone https://github.com/Mazgagzam/Lu_Bot.git
cd Lu_Bot
docker build -t lu-bot .
docker run -d lu-bot
```
